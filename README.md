# Node

Node.js application template.

## Prerequisites

- [Node.js](https://nodejs.org)

## Setup

Install project dependencies:

```console
npm install
```

## Run

Start application:

```console
npm start
```

Get server health:

```console
curl --location --request GET 'http://localhost:3000/health'
```

## Test

Run code style, unit and integration tests:

```console
npm test
```
